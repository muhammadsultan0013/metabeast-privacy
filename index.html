Privacy Policy — MetaBeast

Last updated: September 1, 2024

MetaBeast is a browser extension designed to generate SEO-friendly titles and keyword lists for Adobe Stock uploads. This page explains what data (if any) MetaBeast handles and how it is used.

Summary

MetaBeast does not collect, store, or share personal information such as names, email addresses, or IP addresses. The extension's functionality relies on two types of credentials handled securely: a one-time access key for activation and user-provided Gemini API keys for AI processing.

Data Handled by the Extension

We do not collect personal data. The extension does not request or store usernames, emails, or other personally identifiable information. The data it handles is limited to:

  1. Access Key: A non-personal, one-time key used to activate the extension.
  2. Gemini API Keys: User-provided API keys required to communicate with Google's Gemini AI.

Access Key Verification

To enable access, MetaBeast sends a non-personal access key to a secure Supabase endpoint to validate the key. Supabase is used only to evaluate the access key status. No personal identifiers are stored or associated with the access key in our system.

User-Provided Gemini API Keys

MetaBeast requires users to add their own Google Gemini API keys to function.

  • Storage: These API keys are stored only on the user's local computer using the secure chrome.storage.local API.
  • Transmission: The keys are sent over a secure HTTPS connection to our server-side function, which then uses them to make requests to the Google Gemini API on the user's behalf.
  • Privacy: We never log, store, or collect user API keys on our servers. Their handling is temporary and only for the purpose of fulfilling the AI generation request.

AI Processing Requests

When you request a title or keyword, the extension sends the image data and a user-provided API key to our secure Supabase Edge Function. This function then makes a request to the Google Gemini API. This process is governed by Google's own API and privacy policies.

Third-Party Services

MetaBeast uses Supabase for access validation and our server-side functions. The core AI processing is done by Google's Gemini API. We do not share or sell user data with any other third parties.
